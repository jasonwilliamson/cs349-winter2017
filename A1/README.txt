Objective of the game: 
    * To eat the fruit/block as many times as possible.
    * As the snake eats, it grows. Stay away from the borders and donâ€™t run
      into your snake body. 
    * Collect as many points as possible! Enjoy!!

Use keypad controls WASD:
    w moves up
    a moves left
    d moves right
    s moves down

Additionally, at any time during gameplay:
    p pauses game
    r restarts game

To compile game,
use command: make

To compile and run game with default arguments,
use command: make run

To run game with default arguments,
use command: ./snake

To run game with customize frame rate and speed,
use command: ./snake FPS SPEED (where FPS and SPEED are ints, see note below)

    * Note: FPS supports range of 1-100
            SPEED supports range of 1-10
