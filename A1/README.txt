Objective of the game: To eat the fruit/block as many times as possible.
As the snake eats, it grows. Stay away from the borders and donâ€™t run
into your snake body. Enjoy!

Use keypad controls WASD:
w moves up
a moves left
d moves right
s moves down

Additionally, at any time during gameplay:
p pauses game
r restarts game

To run game with default arguments,
use command: make run

To customize frame rate and speed,
use command: ./snake FPS SPEED (where FPS and SPEED are int values)
