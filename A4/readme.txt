Assignment 4 YouTube Video Gallery. It includes the following files:

Makefile:               'make run' builds and executes the sample             
Main.java:              initializes model and view.
View.java               main window, includes tool bar.           
Canvas.java:            a drawing surface to hold all of the video panels.   
VideoPanel.java:        formatted view displaying video info.
VideoDataType.java:   	serializable data type to store each video's info.
ModelYouTube.java       model using YouTuble api
MyFlowLayout.java       custom extended flowlayout to wrap grid items 
Observer.java


Build environment for A3.
For macOS version 10.12.3, running Java version 8 v121
Tested on linux student environment
